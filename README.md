# EchoEchoFrontend

Frontend prototype and testing environment for **EchoEcho**, an AI-powered songwriting and music inspiration platform.

This repository contains the user interface, page workflows, and frontend logic used to test and validate the user experience of the EchoEcho ecosystem. It serves as a standalone frontend implementation for experimenting with UI/UX, navigation flows, and integration points with backend AI services.

## Purpose

The goal of this repository is to:

* Test frontend features and user flows
* Validate UI/UX designs
* Prototype new functionality before backend integration
* Experiment with application architecture and page navigation
* Serve as a frontend playground for the EchoEcho platform

## Features

* Home Page
* User Authentication Interface
* Dashboard
* Song Generation Workflow
* Output & Results Page
* Modular TypeScript Architecture
* Responsive Design

## Tech Stack

* TypeScript
* Vite
* HTML5

* JavaScript

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/SachinManral/EchoEchoFrontend.git
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
